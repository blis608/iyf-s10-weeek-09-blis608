## 🚀 Installation & Setup

### Clone the repository

```bash
git clone https://github.com/blis608/iyf-s10-week-09-blis608.git
```

### Navigate into the project folder

```bash
cd iyf-s10-week-09-blis608
```

### Install dependencies

```bash
npm install
```

### Run the development server

```bash
npm run dev
```
