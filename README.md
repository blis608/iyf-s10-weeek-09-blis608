## Project Description
This project contains the Week 09 daily challenges and exercise solutions.  
The project focuses on React concepts including Effects, Data Fetching, Routing, React Patterns, and Styling.  
It was built using Vite and React Router with reusable hooks and shared components.

## Implementation Notes
- The app was built using Vite with React and React Router.
- Custom hooks implemented:
  - `src/hooks/useFetch.js`
  - `src/hooks/useLocalStorage.js`
  - `src/hooks/useToggle.js`
  - `src/hooks/useForm.js`
- Shared components are stored under:
  - `src/components/shared/`
- Pages and routing are implemented across:
  - `src/pages/`
- Loading and error states handled using:
  - `src/components/LoadingSpinner.jsx`
  - `src/components/ErrorMessage.jsx`
