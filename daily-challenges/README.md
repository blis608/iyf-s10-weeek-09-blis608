## Implementation notes

- The app was built using Vite with React and React Router.
- Custom hooks implemented:
  - `src/hooks/useFetch.js`
  - `src/hooks/useLocalStorage.js`
  - `src/hooks/useToggle.js`
  - `src/hooks/useForm.js`
- A reusable shared component library lives under `src/components/shared/`.
- Pages and routing are implemented across `src/pages/`.
- Loading and error states are handled with `src/components/LoadingSpinner.jsx` and `src/components/ErrorMessage.jsx`.
